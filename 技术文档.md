# FlowSync 协同任务管理系统 - 技术文档

## 1. 项目概述

FlowSync 是一个基于 Spring Boot + Vue 3 的前后端分离协同任务管理系统，支持项目创建、任务分配、进度追踪、AI 任务拆解等功能。

---

## 2. 系统架构

### 2.1 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| **前端** | Vue 3 + Element Plus | Vue 3.3+ |
| **后端** | Spring Boot 3.x | 3.x |
| **数据库** | MySQL | 8.x |
| **ORM** | MyBatis-Plus | 最新 |
| **AI 接口** | DeepSeek API | deepseek-chat |
| **构建工具** | Maven / Vue CLI | - |

```
浏览器 (Vue 3) 
    ↓ 请求 (axios) 
    ↓ /api/* 
Spring Boot 后端 
    ↓ 查询 
MyBatis-Plus → MySQL
    ↑
DeepSeek AI API (任务拆解)
```

### 4.1 ER 图

### 4.2 核心表结构

| 表名 | 说明 | 核心字段 |
|------|------|---------|
| `sys_user` | 用户表 | id, username, password, real_name, role, phone, email |
| `project_info` | 项目表 | id, name, description, status, priority, owner_id, start_date, end_date |
| `project_member` | 项目成员表 | id, project_id, user_id, role (leader/member) |
| `task_info` | 任务表 | id, project_id, title, assignee_id, status, priority, due_date, description |
| `task_summary` | 总结表 | id, project_id, content, created_by, updated_by, create_time, update_time |

## 5. API 接口文档

### 5.1 项目接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/projects` | 获取项目列表（管理员全部，非管理员按用户过滤） |
| POST | `/api/projects` | 创建项目 |
| PUT | `/api/projects` | 更新项目 |
| DELETE | `/api/projects/{id}` | 删除项目（级联删除任务、总结、成员） |

### 5.2 任务接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/tasks` | 获取任务列表 |
| POST | `/api/tasks` | 新建任务 |
| PUT | `/api/tasks` | 更新任务 |
| PUT | `/api/tasks/{id}/status` | 更新任务状态 |
| DELETE | `/api/tasks/{id}` | 删除任务 |

### 5.3 总结接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/summaries` | 获取总结列表 |
| POST | `/api/summaries` | 新增总结 |
| PUT | `/api/summaries/{id}` | 更新总结 |
| DELETE | `/api/summaries/{id}` | 删除总结 |

### 5.4 用户接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/users` | 获取用户列表 |
| GET | `/api/users/with-projects` | 获取用户及其项目角色 |
| POST | `/api/users/update-password` | 修改密码 |
| POST | `/api/users/login` | 用户登录 |

### 5.5 AI 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/ai/generate` | 调用 DeepSeek AI 生成任务拆解方案 |
| POST | `/api/ai/import` | 导入 AI 生成的任务 |

## 6. 本地启动指南

### 6.1 环境要求

- JDK 17+
- Maven 3.6+
- Node.js 18+
- MySQL 8.0+

### 6.2 数据库初始化

```sql
CREATE DATABASE flowsync_simple CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 6.3 启动后端

```bash
cd backend
mvn spring-boot:run
```

后端启动在 `http://localhost:8080`

### 6.4 启动前端

```bash
cd frontend
npm install
npm run serve
```

前端启动在 `http://localhost:8081`

## 7. 权限体系

### 7.1 角色层级

```
系统管理员 (sys_user.role = 'leader')
  └── 可管理所有项目、所有任务、所有总结
    
项目负责人 (project_member.role = 'leader')
  └── 可管理自己负责项目的任务、拆解任务
    
普通成员 (project_member.role = 'member')
  └── 可查看自己参与的项目，可勾选完成自己的任务
```

### 7.2 权限矩阵

| 功能 | 系统管理员 | 项目负责人 | 普通成员 |
|------|-----------|-----------|---------|
| 查看所有项目 | ✓ | - | - |
| 创建/编辑项目 | ✓ | - | - |
| 管理项目任务 | ✓ | 仅自己负责的项目 | - |
| AI 任务拆解 | ✓ | 仅自己负责的项目 | - |
| 管理所有总结 | ✓ | - | - |
| 管理自己的总结 | ✓ | ✓ | ✓ |
| 查看成员列表(全部) | ✓ | - | - |
| 查看成员列表(参与项目) | - | ✓ | ✓ |

## 8. 常见问题

### 8.1 数据库中文乱码

**现象：** 项目描述、优先级等字段显示乱码

**原因：** MySQL 数据库全局编码为 GBK

**解决：** 在 `application.yml` 中添加 HikariCP 连接初始化 SQL

```yaml
spring:
  datasource:
    hikari:
      connection-init-sql: SET NAMES utf8mb4
```

### 8.2 跨域问题

**现象：** 前端请求后端接口报 CORS 错误

**解决：** 后端已配置 `CorsConfig` 允许跨域请求，前端使用 Vite 代理转发 `/api` 请求
